🎯 PASO 1: CREAR EJECUTABLE (.EXE)
════════════════════════════════════════════════════════════════

1. Ejecuta:
   crear_exe.bat

2. Espera 2-3 minutos

3. Resultado:
   dist\PokeReport-Launcher.exe  (50-80 MB)

4. ¡Listo!
   Ahora puedes compartir SOLO ese archivo .exe

════════════════════════════════════════════════════════════════

🌐 PASO 2: ALOJAR EN GITHUB (GRATIS)
════════════════════════════════════════════════════════════════

1. CREAR REPOSITORIO:
   • Ve a: https://github.com/new
   • Nombre: pokereport-launcher
   • Public (para que todos descarguen)
   • Create repository

2. SUBIR ARCHIVOS:
   
   En la carpeta del launcher ejecuta:
   
   git init
   git add .
   git commit -m "Launcher v3.4.0"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/pokereport-launcher.git
   git push -u origin main

3. CREAR RELEASE:
   • Ve a tu repo en GitHub
   • Click "Releases" → "Create a new release"
   • Tag: v3.4.0
   • Title: PokeReport Launcher v3.4.0
   • Description: Copia changelog de version.json
   • Arrastra el archivo .exe a "Attach binaries"
   • Click "Publish release"

4. ¡LISTO!
   Tu launcher ya está en GitHub

     "download_url": "https://...",

       "Arreglo de bug 2"
     ]
   }

2. ACTUALIZA pokereport_launcher.py:

3. CREA NUEVO .EXE:

   git add .
   git commit -m "v3.5.0 - Mejoras"
   git push

⚙️ CONFIGURACIÓN PARA UPDATES AUTOMÁTICOS:
════════════════════════════════════════════════════════════════

1. Edita updater.py línea 14:
   self.version_url = "https://raw.githubusercontent.com/TU_USUARIO/pokereport-launcher/main/version.json"
   
   (Reemplaza TU_USUARIO con tu usuario de GitHub)

2. Edita version.json línea 3:
   "download_url": "https://github.com/TU_USUARIO/pokereport-launcher/releases/latest/download/PokeReport-Launcher.exe"

3. Integra updater.py al launcher
   (Ya está preparado el código en GUIA_DISTRIBUCION_Y_UPDATES.md)
